如何安装NorthstarCN

首先，NorthstarCN客户端只存在于PC平台，且 需要您拥有并下载了合法的《泰坦陨落2》游戏。

  1. 下载 最新版本NorthstarCN安装包
  2. 将安装包内所有文件解压到您的 《泰坦陨落2》 安装目录. 这个目录的位置取决于您用于安装游戏的平台,以及是否设置了自定义的安装目录。
    • 对于Steam平台 - 右键您游戏库中的 《泰坦陨落2》 > 属性 > 本地文件 > 浏览本地文件
      常见路径: C:\Program Files (x86)\Steam\steamapps\common\Titanfall2
    • 对于Origin平台 - 点击橘子主程序左上角的 Origin -> 应用程序设置 -> 安装与储存 -> 在您的电脑上 -> 游戏库位置
      进入游戏库位置中显示的目录，并将找到名为 “Titanfall2” 的文件夹。
      常见路径: C:\Program Files (x86)\Origin Games\Titanfall2
    • 对于EA桌面应用程序 - 您可以通过一下设置找到游戏库位置： 设置 -> 下载
      进入游戏库位置中显示的目录，并将找到名为 “Titanfall2” 的文件夹。
      常见路径: C:\Program Files\EA Games\Titanfall2
  3. 双击 NorthstarLauncher.exe 来启动NorthstarCN。
    • 在您第一次运行NorthstarCN时，将会有一个提示框请求Origin Token访问权限。 请点击“是”。 (您可以在Mods选项中更改这项设置)
  4. 点击 启动NorthstarCN
  5. 在多人游戏页面中，您可以点击服务器浏览器来寻找中意的社区服务器进行游玩。
